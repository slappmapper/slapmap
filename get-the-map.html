faq

can i actually hide a link on a static site?

You can't make it impossible to view the source, but redirecting buyers back here keeps the flow smooth. The real security lives in Google My Maps — only people with the URL can copy it. Rotate the link occasionally if you need to.

what if someone shares the link?

You can revoke the old My Maps link at any time. Create a fresh duplicate, update the link in this file, and only paying folks will have the latest copy.

do i need code to run payments?

Nope. Stripe Payment Links, Gumroad, or Lemon Squeezy all host checkout for you. This page simply forwards people there and handles the success redirect.
